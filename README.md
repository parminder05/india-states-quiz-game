# 🇮🇳 India States Quiz Game

An interactive quiz game built using Python Turtle and Pandas where users guess Indian states on a map.

---

## 🎮 Features
- 🗺️ Interactive India map
- ✍️ Type and guess state names
- ✅ Correct answers appear on the map
- ❌ Tracks missed states
- 📁 Exports missed states to CSV for learning

---

## 🛠️ Technologies Used
- Python
- Turtle Graphics
- Pandas
